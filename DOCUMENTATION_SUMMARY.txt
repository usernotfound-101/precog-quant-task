FORMULA HIGHLIGHTS
================================================================================

Key Equations Documented:

1. RSI = 100 - (100 / (1 + RS_t))
   where RS_t = AvgGain_t / AvgLoss_t

2. Rogers-Satchell Volatility:
   RS_t = ln(H/O)·ln(H/C) + ln(L/O)·ln(L/C)

3. Kalman Gain:
   K_t = P_{t|t-1} / (P_{t|t-1} + R)

4. Trading Score:
   TradeScore_t = MA_3(pred_return_t / (pred_risk_t + ε))

5. Sharpe Ratio:
   Sharpe = (μ_r - r_f) / σ_r × √252

6. Maximum Drawdown:
   MDD = min_t (PV_t / max_{s≤t} PV_s - 1)

7. Position Weight:
   w_i = (1/σ_i) / Σ(1/σ_j)

8. Hedge Ratio:
   β = Cov(S1^log, S2^log) / Var(S2^log)
